Problem Description

Given an integer array nums, return all the triplets [nums[i], nums[j], nums[k]] such that i != j, i != k, and j != k, and nums[i] + nums[j] + nums[k] == 0.

Notice that the solution set must not contain duplicate triplets.

Example:

Input: nums = [-1,0,1,2,-1,-4]

Output: [[-1,-1,2],[-1,0,1]]

Explanation: nums[0] + nums[1] + nums[2] = (-1) + 0 + 1 = 0.
nums[1] + nums[2] + nums[4] = 0 + 1 + (-1) = 0.
nums[0] + nums[3] + nums[4] = (-1) + 2 + (-1) = 0.
The distinct triplets are [-1,0,1] and [-1,-1,2].

Algorithm

We use a combination of sorting and two-pointers approach:

  1. Sort the array to handle duplicates and enable two-pointers technique
  2. For each element, use two pointers to find pairs that sum to the negative of current element
  3. Skip duplicates to avoid duplicate triplets

Complexity Analysis

Time Complexity: O(N²)
O(N log N) for sorting + O(N²) for nested loops with two pointers

Space Complexity: O(1)
Not counting the space for the output array
