Problem Description

Given two strings s and t, return true if t is an anagram of s, and false otherwise.

An Anagram is a word or phrase formed by rearranging the letters of a different word or phrase, typically using all the original letters exactly once.

Example 1:

Input: s = "anagram", t = "nagaram"

Output: true

Explanation: Both strings contain the same characters with the same frequencies.


Example 2:

Input: s = "rat", t = "car"

Output: false

Explanation: Different characters: 'r','a','t' vs 'c','a','r'.

Complexity Analysis

Time Complexity: O(N)
We iterate through both strings once. Each character operation is O(1).

Space Complexity: O(1)
We use a fixed-size array of 26 elements for lowercase English letters.

Java Solution

class Solution {
    public boolean isAnagram(String s, String t) {
        if (s.length() != t.length()) {
            return false;  // Different lengths = not anagram
        }
        
        int[] count = new int[26];  // Count array for a-z
        
        for (int i = 0; i < s.length(); i++) {
            count[s.charAt(i) - 'a']++;  // Increment for s
            count[t.charAt(i) - 'a']--;  // Decrement for t
        }
        
        for (int i = 0; i < 26; i++) {
            if (count[i] != 0) {
                return false;  // Non-zero = different frequencies
            }
        }
        
        return true;  // All counts are zero = anagram
    }
}

Intuition

The key insight is that anagrams have identical character frequencies. We use a clever counting technique with a single array.

  1. Length Check: If strings have different lengths, they can't be anagrams.
  2. Count Array: Use an array of size 26 for lowercase letters a-z.
  3. Simultaneous Counting: For each position, increment count for string s and decrement for string t.
  4. Balance Check: If strings are anagrams, all counts will be exactly zero.
  5. Final Verification: Any non-zero count means different character frequencies.
